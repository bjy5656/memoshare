정규화 
삽입/삭제/갱신 이상현상(anomaly)을 제거하기 위한 작업 
삽입이상 : 데이터를 추가할 때 값이 없는 데이터도 추가되는 것 
갱신이상 : 값이 중복되는 데이터가 여러 개 있을 경우 하나의 데이터를 갱신하면 다른 데이터들과 불일치 발생 
삭제이상 : 특정 데이터를 삭제할 때 같은 행에 있는 데이터도 삭제되는 문제 

1차정규화(1NF)
- 모든 칼럼의 값은 원자값(단일값)을 갖도록 분해 

2차정규화(2NF) 
- 1차 정규화를 만족 
- 기본키가 복합키일 경우 부분함수 종속을 제거 
- 부분함수 종속 : 복합키를 이루는 키 중 일부 키에만 종속되는 경우 

3차정규화(2NF) 
- 2차 정규화를 만족
- 이행적 종속을 제거 
- 기본키가 아닌 칼럼에 비주요칼럼이 종속되는 경우 해당 칼럼들을 새 테이블로 분리 

* 종속관계는 함수 관계가 성립되는 것이다. 특정 칼럼이 다른 칼럼을 종속시킨다고 해서 그 칼럼이 항상 기본키 자격을 갖추는 것은 아니다. 
* 다대다 관계면 중단 테이블이 필요
